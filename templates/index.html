Toolforge Quick Guide

Welcome to Deployr Dashboard! This interface connects directly to your local file system to read SSH keys and control the Wikimedia Toolforge environment.

Configuring Access: Ensure your registered public SSH key is uploaded to your Wikitech developer profile and that you provide the corresponding private key file path (e.g. ~/.ssh/id_rsa) on the left.

Deploying Apps: Paste any public git repository or direct zip archive link into the Deploy tab to clone, package, upload, configure uWSGI, build venv and start the Toolforge webservice in one step.

Active Configuration Directory: ~/.toolforge_config.json
